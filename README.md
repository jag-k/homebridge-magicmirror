# Homebridge MagicMirror<sup>2</sup> Plugin

> Add your MagicMirror in Apple HomeKit!

## Features

- Turn On/Off monitor in [MagicMirror]
- Change brightness interface
- Turn On/Off to display each MagicMirror Module
